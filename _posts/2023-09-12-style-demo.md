### Code Block

This can display a block of `code`.

```
<pre>
<code>
  body {
    color: var(--text);
    background: var(--bg);
    font-size: 1.15rem;
    line-height: 1.5;
    margin: 0;
  }
</code>
</pre>
```


### Details/Summary

The `details` of this element are hidden behind a `summary`.

  <summary>Summary</summary>
  <p>Some text.</p>
</details>
